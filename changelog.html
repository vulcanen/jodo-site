更新日志

每个版本都在这里留个脚印。

v1.0.0 · 2026-05-16

首个公开发布版本（Android）。

任务

  • 任务池 / 今天 / 计划 / 沉睡 / 已完成 / 归档 / 最近删除 全套状态机
  • 标题、描述、优先级、标签、重复（每日 / 每周 / 每月）、默认专注模式
  • 中央 + 快速新增；长按 / 左滑出菜单（编辑 / 归档 / 删除）
  • 完成 / 归档 / 删除全部带 6 秒撤销窗
  • 30 天未动转为「沉睡」；删除任务 30 天后自动物理清理
  • 周历 + 月历视图；标题和描述全文搜索

专注

  • 番茄钟 / 正计时 / 自定义 三种模式
  • 开始 15 秒内取消不计入统计
  • 可绑定今日任务，专注时长回写任务累计
  • 全屏运行态自动隐藏底部导航和 FAB

统计

  • 连续打卡天数 + 历史最长
  • 当月完成情况热图、当周专注分钟柱状图、当月完成率

设置

  • 浅色 / 深色 / 跟随系统 三套主题
  • 中文 / English / 跟随系统 三种语言
  • 自定义默认番茄时长与休息时长
  • 每日提醒（本地通知，无网络）

数据

  • JSON 导出 + 分享到任意应用
  • JSON 导入（合并 / 替换 两种模式）
  • 所有数据保存在本机 SQLite，不联网、不收集、不上传

平台

  • Android 11 (API 30) 及以上
  • 包名 com.goose.jodo
  • iOS 待后续
