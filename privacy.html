隐私政策

生效日期：2026-05-16

一、概述

Jodo 是一款本地优先的「任务 + 专注」生产力工具，完全不联网、不收集任何用户数据。 本隐私政策说明应用在设备上会处理哪些数据，以及我们承诺不做什么。

二、应用在本机会处理的数据

Jodo 只在你的设备上存储以下信息，用于提供核心功能：

  • 你创建的任务（标题、描述、优先级、标签）
  • 任务状态（收件箱 / 今日 / 已完成 / 沉睡 / 归档 / 最近删除）
  • 专注会话记录（开始/结束时间、模式、关联任务）
  • 任务的创建时间、更新时间、完成时间
  • 外观偏好与设置（主题、语言、默认番茄时长、默认休息时长、提醒开关与提醒时间）

以上所有数据都保存在设备本地的 SQLite 数据库中。 不会上传到任何服务器，也不会发送给任何第三方。

三、我们明确不做的事

  • 不收集姓名、邮箱、电话、设备 ID 等任何身份信息
  • 不做任何使用统计、埋点、行为分析
  • 不集成任何第三方 SDK（广告、统计、推送、崩溃分析等）
  • 不请求网络访问权限
  • 不请求通讯录、相册、位置、通话、相机、麦克风等敏感权限
  • 不集成任何 AI / 大模型服务（你的任务文本永远不会被发到外部）
  • 不在后台收集或上传任何数据

四、应用请求的权限

Jodo 运行时不主动请求任何敏感权限。以下权限仅在用户主动发起操作时才触发：

文件读取 用户点击「导入数据」时，读取选定的 JSON 备份文件。
文件写入 / 分享 用户点击「导出数据」时，生成 JSON 备份到选定位置。
通知 用户首次启用「每日提醒」时，用于本地通知（每日提醒 / 专注完成 / 休息结束）。只本机产生，不联网。

权限仅在当次操作或用户明确开启的提醒功能中使用，不会用于其它目的。

五、数据的备份与迁移

应用内置「导出数据」功能，生成一份包含你全部任务、专注会话、标签的 JSON 文件，完全由你自己掌控。你可以：

  • 保存到本地文件管理器
  • 通过微信 / 邮件 / AirDrop / 其他应用分享
  • 导入到另一台设备上的 Jodo

导出文件不包含任何身份信息，只有你输入过的任务 / 专注数据本身。

六、数据的删除

  • 归档任务：左滑归档，任务从主列表移除但数据永久保留，可随时恢复。
  • 删除任务：左滑删除，任务进入「最近删除」回收站，30 天内可恢复；过期后系统自动物理清理。
  • 立即彻底删除：在「最近删除」中可对单条任务执行「永久删除」，数据立即从设备上抹除，不可恢复。
  • 全部清除：卸载应用后所有本地数据随系统自动删除；也可使用系统设置的「清除应用数据」手动清空。

七、数据安全

由于所有数据都保存在本机，安全性取决于你的设备本身。建议：

  • 启用设备锁屏（指纹 / 面容 / PIN 码）
  • 不要在 root / 越狱设备上保存敏感任务内容
  • 定期使用「导出数据」做备份，并把备份文件放到安全位置

八、儿童隐私

Jodo 不针对任何特定年龄段，但因为不收集任何数据，也就不存在儿童个人信息的收集与处理。

九、政策变更

如果这份政策有更新，会同步修改应用内的「隐私政策」页面和本页面，并把新的「生效日期」标注在文首。 对应版本的应用首次启动时可能会显示简要提示。
