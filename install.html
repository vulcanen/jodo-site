安装说明

从下载到打开 App，大概 1 分钟。

步骤

  1. 点击 首页下载按钮，浏览器直接下载 Jodo-v1.0.4.apk 到手机。
  2. 下载完成后，在文件管理器或浏览器的"已下载"里点击这个 APK。
  3. 按系统提示，允许「当前来源」安装应用。不同手机品牌位置略有差异，但都是一次性的开关。
  4. 安装完成后，回到桌面打开 Jodo。
  5. （可选）安装完后，可以把那个"允许未知来源"的开关再关掉。

关于"未知来源"提示

Android 对非应用商店安装包会显示安全提示，这是系统的正常保护机制。 请只从 Jodo 官网链接下载，并核对版本号与 SHA-256。

验证文件完整性（可选）

如果你想确认 APK 没有在传输中被替换，可以核对 SHA-256。 在 Windows / PowerShell 中： 

Get-FileHash -Algorithm SHA256 Jodo-v1.0.4.apk

在 macOS / Linux 中： 

shasum -a 256 Jodo-v1.0.4.apk

结果应该和首页下载区显示的 SHA-256 完全一致。

升级到新版本

下载新版本 APK，按上面的步骤再装一次即可，原有数据不会丢失（同包名同签名直接覆盖安装）。 如果想换设备迁移数据，使用 App 内「我的 → 导出数据」生成 JSON 备份，到新设备「我的 → 导入数据」即可。

卸载与数据清理

直接卸载 App，系统会自动清掉所有本地数据。 你也可以在系统设置 → 应用 → Jodo → 存储 → "清除数据" 手动清空，不卸载也能恢复初始状态。
